# SocialEngine

SocialEngine is a little base of a social network designed with laravel & react + bootstrap

Features:
->Templating
->Auth
->Profil/Custom
->Add friends
->Search
->Real time conversation
->Notifications (push)
->Vues/Likes/Reputation/Badges
->Comments
->Presence system
->geolocalisation search
+Robot/Friend
